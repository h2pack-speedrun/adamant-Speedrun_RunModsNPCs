# Run Mods: NPCs

Control NPC spawn and route-adjacent behavior for speedrun-focused run shaping.

Part of the [Speedrun modpack](https://thunderstore.io/c/hades-ii/p/adamant/Speedrun_Core/).

## What It Does

Run Mods: NPCs changes NPC and route-adjacent behavior that matters for attempt consistency.

The module currently includes:

- Disable Arachne Pity
  Turns off Arachne pity.
- Disable Selene Before First Boon
  Prevents Selene from appearing before the first boon or hammer is obtained.
- Force Arachne Spawn
  Forces Arachne to appear.
- Force Medea Spawn
  Forces Medea to appear.
- Prevent Echo Scam
  Blocks both Fields minibosses from spawning at room 3 so Echo cannot be crowded out there.

Use it when you want tighter control over NPC-related route variance without changing the broader world structure.

## Installation

Install via r2modman.

This module is usually installed as part of the full [Speedrun modpack](https://thunderstore.io/c/hades-ii/p/adamant/Speedrun_Core/), where it appears in the shared Speedrun UI with the other speedrun-focused modules.
